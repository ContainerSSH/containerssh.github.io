    1.0.0: Ready for Production¶

    • Web client
    • SSH agent forwarding
    • In-Kubernetes authentication and configuration
    • Signed package repo for DEB, RPM and APK

    0.5.0: SSO¶

    • Keyboard-interactive authentication
    • SSH port forwarding
    • Kerberos/GSS-API authentication
    • Support WebAuthn
    • Pass metadata from the auth webhook to the config server
    • Health check endpoint
    • ARM / 32 bit support
    • containerssh does not cleanup all pods
    • Pulling image from private repositories on Docker
    • Port Forwarding Seems Broken
    • SSH Server only offering one kex algo, config options aren't affecting this.
    • ContainerSSH does not work with Jetbrain remote development
    • ContainerSSH Metrics are not Prometheus Compatible

    Future¶

    • PAM authentication

    Ideas¶

    • File manager
    • Support Gitlab CI runners
    • Direct shell backend
    • Add support for UDP forwarding
    • Launch VMs
    • Support ECS exec
    • Nomad backend support?
    • Antivirus Backend for file transfers (scp/sftp)
