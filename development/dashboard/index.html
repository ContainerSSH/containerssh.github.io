    1.0.0: Ready for Production¶

    • Web client
    • SSH port forwarding
    • SSH agent forwarding
    • In-Kubernetes authentication and configuration
    • Signed package repo for DEB, RPM and APK

    0.5.0: SSO¶

    • Keyboard-interactive authentication
    • Support WebAuthn
    • Pass metadata from the auth webhook to the config server
    • ARM / 32 bit support

    0.4.2: Health check¶

    • Health check endpoint

    Future¶

    • Stopping ContainerSSH does not remove containers
    • Antivirus Backend for file transfers (scp/sftp)
    • PAM authentication

    Ideas¶

    • File manager
    • Support Gitlab CI runners
    • Direct shell backend
    • Kerberos/GSS-API authentication
    • Add support for UDP forwarding
    • Launch VMs
    • Support ECS exec
